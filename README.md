### 📌 Featured Projects

#### 🌾 Panruti Cashew Farmer Marketplace *(Full-Stack + AI)*
*A full-stack e-commerce and farmer marketplace web application for selling premium Panruti cashews directly from farmers to customers. Features separate Admin and Farmer portals, an AI-powered Cashew Quality Checker where farmers upload images with live GPS tags — the system predicts a quality score and sends results to the admin for approval or rejection. Once approved, products appear in the marketplace where customers can browse, add to cart, and place orders online. Built to empower farmers with technology, ensuring quality assurance, transparency, and fair trade.*
> **Tech:** React, Node.js, Express.js, PostgreSQL, Firebase Authentication, AI Image Analysis
